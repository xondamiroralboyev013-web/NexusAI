## üîß **Installation Steps**

### **Step 1: Download**
Choose one method above to download NexusAI

### **Step 2: Extract**
```bash
unzip NexusAI-Complete-Source.zip
cd NexusAI
```

### **Step 3: Install Dependencies**
```bash
npm install
```

### **Step 4: Get API Key** (Free)
- Go to: https://console.groq.com
- Sign up (free, no credit card)
- Copy your Groq API key

### **Step 5: Choose Your Platform**

#### **A. Web Version (Browser)**
```bash
1. Open: www/app.html in Chrome/Firefox
2. Add your Groq API key in Settings
3. Start using NexusAI!
```

#### **B. Mobile App (Android)**
```bash
1. Install Android SDK (if not installed)
2. Set ANDROID_HOME environment variable
3. Run: cordova build android --release
4. Install APK on your Android device
```

#### **C. Local Server**
```bash
npm run serve
# Opens at http://localhost:8080
```
